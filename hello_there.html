To add a file in the repository:
git add $file$
To add a commitment
git commit -m "Your message of changes here"
Getting the git status you can see how often commitments have you done so far:
git status
Adding the comand to push, you upload the data you have done:
git push -u
The commit -m is just to commit a unique file at a time, if u want to commit a lot of files, use:
git commit -am "message here"
And if your are working with more people, you can download the new code:
git pull

In a conflict(A partner edits a part and you also edits that part, in a different way), github would automatically put a text like:
Code hasn't change
<<<<<
/Your code part modified/

To see who have made edits yo can run:
git log
Every commit has an id, this id can be used to reset the repository:
git reset "id"

U can get a lot of branches, adding new futures and etc, the "head" is referred as the main branch, where is the main code.
typing
git branch
You get the branch you currently are. To get a new branch you can text
git checkout -b "Name Of the new branch"
